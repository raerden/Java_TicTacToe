# 🎮 Tic-Tac-Toe Game

A web-based implementation of the classic Tic-Tac-Toe game built with Spring Boot framework.

## ✨ Features
- Play Tic-Tac-Toe in your browser
- Clean MVC architecture
- RESTful API for game operations
- Session management for multiple games

## 🛠️ Technology Stack
- **Backend:** Java 17+, Spring Boot, Spring MVC
- **Build Tool:** Maven/Gradle
- **Server:** Embedded Tomcat
- **Frontend:** Thymeleaf (or your frontend technology)

## 🚀 Getting Started
1. Clone the repository
2. Run `mvn spring-boot:run` or `./gradlew bootRun`
3. Open `http://localhost:8080` in your browser

## 📡 API Endpoints
- `GET /` - Main game page
